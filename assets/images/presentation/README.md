## Téléchargement depuis Figma

Les images et icônes sont disponibles dans le design Figma à l'URL fournie.
Pour les télécharger :

1. Ouvrez le design dans Figma
2. Sélectionnez chaque image/icône
3. Clic droit > Export > PNG (ou SVG pour les icônes)
4. Placez les fichiers dans les dossiers correspondants

## Note

En attendant d'avoir les vraies images, vous pouvez utiliser des images placeholder de la même taille.
